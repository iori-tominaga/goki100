# 要件定義書 — goki100

タイトル：**「ゴキブリは1匹みたら100匹いると思え」**

## 1. コンセプト

家の中でキモかわゴキブリを増殖させ、**100匹**到達を目指すリアルタイム育成くそげー。
遊ぶことがメインの、笑えて理不尽なゲーム。

## 2. キャラクター

- **ゴキブリ**：デフォルメされ気持ち悪くならない造形。顔と手足は人間、体はゴキブリの着ぐるみを着ている感じ。
- 塊魂リスペクトの原色ポップな世界観に馴染ませる。

## 3. ゲームルール

| 項目 | 内容 |
|---|---|
| 開始 | 1匹のゴキブリが家に侵入するところから |
| 操作 | 十字キー（矢印 / WASD）で家の中を移動 |
| 収集 | エリア内にランダム点在する食べ残し・段ボールを拾う → ゲージ蓄積 |
| 増殖 | ゲージ満タンで1匹増殖 |
| 仲間AI | 増えた個体は操作不要。自律的に家を動き回り食料を回収 |
| 危険 | ゴキブリホイホイ等のトラップ、飼い猫の襲撃、家主のゴキジェット等の死亡アクションが多数点在 |
| ゴール | ゴキブリ100匹到達 |

## 4. グラフィック要件

- 3D。塊魂リスペクトの原色ポップ・ローポリ。
- ゴキが小さい世界観のため、家具などのオブジェクトは**特大サイズ**で生成。
- 斜め後ろから追う陽気な追従カメラ。

## 5. 技術要件

- Three.js（ビルド不要 / importmap）。
- ゲームロジックと描画を分離し、将来 Babylon.js 等へ差し替え可能にする。
- 静的サイトとして GitHub Pages で公開できること。

## 6. スコープ（フェーズ）

- Phase 0: 3D空間・特大家具・ゴキ1匹移動・追従カメラ ← 最初のマイルストーン
- Phase 1〜4: 収集→増殖→仲間AI→死亡アクション→100匹ゴール
