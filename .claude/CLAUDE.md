# CLAUDE.md — goki100

## プロジェクト概要

タイトル：**「ゴキブリは1匹みたら100匹いると思え」**

家の中でゴキブリ（キモかわデフォルメ：顔と手足は人間＋ゴキ着ぐるみ）を増殖させ、
100匹到達を目指すリアルタイム育成ゲーム。ブラウザ上で動く3Dくそげー。

### コアループ
1. 十字キー（矢印 / WASD）で家の中を移動
2. 点在する食べ残し・段ボールを拾ってゲージを貯める
3. ゲージ満タンで1匹増殖 → 増えた個体は**自律AI**で勝手に食料回収
4. ゴキブリホイホイ・飼い猫・家主のゴキジェット等の死亡アクションを回避
5. **100匹**到達でクリア

---

## デザイン方針

**「塊魂（かたまりダマシー）」リスペクト**。

- 原色ポップでビビッドな配色。キモさより「楽しい・陽気」を優先
- 家具・小物はローポリ＋**Canvasテクスチャ**（牛乳パック・段ボール・サイコロ・缶・本・お菓子）
- ゴキが小さい世界観 → 家具・**特大の人間（家主/おばあちゃん＝障害物）**は特大サイズで生成
- キャラのすぐ後ろ・地面に近い低い三人称追従カメラ
- ゴキは「人間がゴキ着ぐるみを着た」造形：大きな殻が背中〜頭上を覆い、人間の顔が前から覗く（gokisample 参考）
- ゴキは二足歩行。移動中は腕と脚を逆位相でスイング

## 技術スタック

| 項目 | 採用技術 |
|---|---|
| 3D描画 | Three.js（ES Modules / importmap, ビルド不要） |
| ゲーム本体 | 素の JavaScript（描画非依存） |
| 公開 | GitHub Pages（src/ を配信元） |

### 最重要の設計思想：レンダラの抽象化

**ゲームロジック（状態）と描画（見た目）を完全に分離する。**

- `src/js/game/` … ゴキブリの位置・ゲージ・増殖数などの状態。Three.js を一切 import しない
- `src/js/render/Renderer.js` … 共通インターフェース（基底クラス）
- `src/js/render/ThreeRenderer.js` … Three.js 実装

将来グラフィックや死亡アクションに不満が出たら、`BabylonRenderer.js` を
同じインターフェースで新規実装し**差し替えるだけ**で済むようにする。
ゲーム本体には手を入れない。

| Phase | 内容 |
|---|---|
| **Phase 0** | 3D空間＋特大家具＋ゴキ1匹表示、十字キーで移動、追従カメラ ← 現在地 |
| Phase 1 | 食べ残し/段ボール収集 → ゲージ → 1匹増殖 |
| Phase 2 | 増えた個体の自律AI（勝手に回収） |

## ディレクトリ構成

```
goki100/
├── .github/workflows/deploy.yml   # GitHub Pages デプロイ（src/ 配信）
├── src/
│   ├── index.html
│   ├── style.css
│   └── js/
│       ├── main.js                # エントリ・ゲームループ
│       ├── game/                  # ★描画非依存のゲームロジック
│       │   ├── config.js          # 定数・ITEMS(生活小物)・GIANTS(特大人間)・VARIANTS(顔)
│       │   ├── state.js           # 状態・小物/人間生成・当たり判定(円形押し出し)
│       │   └── input.js
│       └── render/                # ★描画実装（差し替え可能）
│           ├── Renderer.js        # 共通インターフェース
│           ├── ThreeRenderer.js   # Three.js 実装（ゴキ着ぐるみ/小物/特大人間）
│           └── textures.js        # Canvas でラベル等を描く小物テクスチャ
└── docs/specs/requirements.md
```

## 起動方法

ビルド不要。ローカルで簡易サーバーを立てて開く（ES Modules は file:// では動かないため）。

```bash
cd goki100/src
python -m http.server 8000
# → http://localhost:8000 を開く
```
